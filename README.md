- java
-  Oracle SQL

## DCTalk 주요 기능
- 카카오 & 페이스북 계정을 이용해 회원가입
- 해쉬 코드를 이용한 시크릿 단톡방
- 이미지, 동영상 첨부 가능
- 닉네임 제외 -> 실명 강제
- 지도 API -> Kakao Map 지도 (좌표) 첨부 가능 -> 개개인별로 목적지 장소로 이동할 수 있는 네비게이션 기능
- 공용 자금 입출금 서비스 및 잔액 확인
- 외부 링크 접속 및 저장
- resistory 구축
- 홈페이지 고객센터 & 관리자 모드
